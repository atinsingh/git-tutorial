### LEARNING GIT 
## start a repo 
```shell
git init 
```
or 
```shell
git init <repo>
```

## check remote attached to repo
```shell
git remote -v
```
# Add a remote
```shell
git remote add origin https://github.com/atinsingh/git-tutorial.git
```

# learn to sync remote
# push change to remote
```shell
git push <remote_name> <branch_name>
```

# pull change
```shell
git pull <remote_name> <branch_name>
```
